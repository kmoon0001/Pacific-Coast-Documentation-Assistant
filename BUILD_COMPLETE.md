## 🚀 Deployment Options

### 1. Vercel (Recommended - Easiest)
```bash
npm install -g vercel
vercel --prod
```
**Time**: 2 minutes  
**Cost**: Free  
**URL**: Automatic HTTPS URL

### 2. Netlify (Also Easy)
```bash
npm install -g netlify-cli
netlify deploy --prod --dir=dist
```
**Time**: 2 minutes  
**Cost**: Free  
**URL**: Automatic HTTPS URL

### 3. Manual Upload
1. Zip the `dist/` folder
2. Upload to any static hosting
3. Configure HTTPS
4. Done!

## 🔑 BYOK (Bring Your Own Key) Setup

### How It Works
1. User opens deployed app
2. App prompts for Gemini API key
3. User enters their free API key
4. Key stored in browser localStorage
5. App uses their key for AI generation

### Getting API Key
**URL**: https://makersuite.google.com/app/apikey

**Free Tier**:
- 60 requests per minute
- 1,500 requests per day
- Perfect for individual use

**No Credit Card Required!**
