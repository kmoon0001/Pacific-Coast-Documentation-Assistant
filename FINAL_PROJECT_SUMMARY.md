## Deployment Instructions

### Pre-Deployment
```bash
# Run all tests
npm run test:run

# Generate coverage report
npm run test:coverage

# Run E2E tests
npm run e2e

# Build for production
npm run build

# Verify build
ls -la dist/
```

### Deployment
```bash
# Deploy to staging
npm run deploy:staging

# Run smoke tests
npm run e2e:staging

# Deploy to production
npm run deploy:prod

# Verify deployment
npm run health-check:prod
```

### Post-Deployment
```bash
# Monitor logs
npm run logs:prod

# Check metrics
npm run metrics:prod

# Verify health
npm run health-check:prod
```
